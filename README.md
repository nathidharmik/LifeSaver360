# LifeSaver360

## Problem Statement

People lose valuable time during emergencies searching for emergency contacts, hospitals and first aid instructions.

## Solution

LifeSaver360 is a mobile-friendly emergency response web application that provides quick access to emergency services, first aid guidance, GPS location, and nearby hospitals.

## Features

• Panic Mode
• SOS
• GPS Location
• Nearby Hospitals
• Share Location
• Emergency Numbers
• First Aid Guides
